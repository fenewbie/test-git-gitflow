Install gitflow

Any update from develop branch, you need to git pull before git push

Delete local branch : git branch -d name_branch

Delete remote branch : git push --delete name name_remote_branch

Delete changes in local and remote branch git fetch --all --prune

PUBLISH BRANCH from local to remote: git add and git commit then git flow feature gitflow name_branch

remove commit : git cherry-pick name commit

Fetch remote branch which doesn't exist in local. You need to quit current branch, go back develop branch and git flow feature pull origin name_branch

Create a feature branch: git flow feature start feature_branch

When finish the feature branch and you will merge this feature into develop branch git flow feature finish feature_branch
